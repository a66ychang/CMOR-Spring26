For my CMOR 438 project, I will be implementing various machine learning algorithms on global emissions and climate-related datasets. I am particularly interested in environmental awareness and hope to eventually pursue a career in climate-tech.

This project is designed to go beyond applying models in isolation by building a data-driven narrative around climate change. Using datasets on CO₂ emissions, methane output, agricultural emissions, and global temperature change, I explore how human activities contribute to environmental impact over time. I then apply both supervised and unsupervised learning techniques to identify patterns in emissions, quantify their relationship with temperature change, and assess differences across countries and sectors.

Finally, I extend this analysis to examine how environmental trends may relate to market behavior, using clean-energy indicators such as Tesla’s stock performance as a proxy for technological and economic response. Overall, this repository aims to demonstrate how machine learning can be used not only for prediction, but also for uncovering structure, generating insights, and better understanding the global dynamics of climate change.
